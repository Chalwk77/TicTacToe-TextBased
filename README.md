# Tic Tac Toe

This is text-based Tic Tac Toe game written in Java.

Features:

- Player vs Computer
- Board sizes: 2x2, 3x3, 4x4, 5x5, 6x6, 7x7, 8x8, 9x9

## Requires [Java 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html) or higher.
